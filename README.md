
# VHS Hospital Website Redesign

## Project Overview

This project is a modern website redesign created for VHS Hospital to improve patient experience through a clean, accessible, and user-friendly interface. The design was created using Figma with a focus on responsive layouts, clarity, and easy navigation.

## Pages Included

* Home Page
* About the Hospital
* Departments
* Doctors / Consultants
* Services & Facilities
* Patient Information
* Contact Page

## Features

* Modern Healthcare UI Design
* Responsive Layout
* Patient-Friendly Navigation
* Clean Typography and Visual Hierarchy
* Accessible and User-Centered Design
* Professional Hospital Branding

## Design Goals

* Improve user experience for patients
* Provide easy access to hospital information
* Create a professional and trustworthy healthcare interface
* Ensure responsive and accessible design

## Tools Used

* Figma
* Canva
